The saved resourcepacks will be in your default resourcepacks folder.

This Mod adds a button next to your resourcepacks button that allows you to download server resourcepacks.

Download on [Modrinth](https://modrinth.com/mod/simple-resourcepack-downloader), [CurseForge](https://www.curseforge.com/minecraft/mc-mods/simple-resourcepack-downloader) or [Github](https://github.com/J0KER2J0KER/SimpleResourcepackDownloader/releases/latest).

# Fabric API
The Fabric API is required for this Mod to work. It can be downloaded [here](https://modrinth.com/mod/fabric-api).
